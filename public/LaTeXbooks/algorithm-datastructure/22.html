3.7 マージソート 20 3.7 マージソート マージソートは、 1. 配列の 2 分割を繰り返して、それぞれの配列の要素数が 1 となるまで細分化する 2. お互いの値を比較して並べ替えながら併合していき、一つの配列に戻す という操作により、全体を整列させる方法である。 昇順マージソート // 配列 A の left から right までの範囲をマージソート function MergeSort(A, B = [], left = 0, right = A.length) { // 配列の大きさが 1 以下であればソート済み if (right - left <= 1){ return A; } // 配列を二等分するため、中央のインデックスを求める var mid = (left + right) / 2; // 要素数が奇数の場合、mid は小数になるので切り捨て mid = Math.floor(mid); // 配列の前半と後半をそれぞれソートする MergeSort(A, B, left, mid); MergeSort(A, B, mid, right); // ソートした配列をマージする return Merge(A, B, left, mid, right); }
